Color Blocks

STEPHEN ALBERTS

Color Blocks is based on a “random walk” algorithm. The algorithm continuously chooses a new direction for the next block to be placed. At the same time, it is walking back and forth within red, green, and blue values, creating the change of color between each block. The color values are applied to the height, width, and depth of the blocks resulting in distinctive shapes for each color.
